Overview of the project

A full-stack Team Task Management web application that allows users to collaborate on projects, manage tasks, and track progress efficiently. Built as a simplified version of tools like Trello/Asana.

✨ Features
1 User Authentication (Signup/Login)
2 Role-based Access (Admin & Member)
3 Project Management (Create & Join Projects)
4 Task Management (Create, Assign, Update Status)
5 Dashboard with task analytics
6 Overdue task tracking
7 Status tracking (To Do / In Progress / Done)
🛠️ Tech Stack
Frontend: HTML, CSS, JavaScript
Backend: Python (Flask)
Database: SQLite / SQLAlchemy
Authentication: Flask-Login
Security: Flask-Bcrypt
Deployment: Replit
